# Task Manager App

A simple full-stack Task Manager application built using
Node.js, Express, MongoDB, and Vanilla JavaScript.

## Features
- Add tasks
- View all tasks
- Delete tasks
- REST API based backend

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
